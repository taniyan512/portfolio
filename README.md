# ギタっぷるとは
ギターのレッスンを受けたい人、レッスンを誰かに提供したい人をマッチングさせるサービスです。  
自己アピールとしてyoutubeの動画を貼り付けることができます。  
気軽に知らない方と会うのが懸念される昨今ですが、ギタっぷるを使って、オンラインで繋がりましょう。  

# ギタっぷるを作った理由  
ギターは、その難しさから初心者のうち9割が辞めると言われており、上達が難しい楽器とされています。そこで上手くなりたいと思い、誰かに教えてもらうことで上達したいと思う人がおられますが、田舎に住んでいる方や、周りにギターを教えて欲しくても、「自分以外にギターを弾いている人がいない」「近くにギターレッスンをしているところがない」「ギターを教えて欲しいけど人に会うのは嫌」といった理由でギターを諦めている人がたくさんいます。そうした悩みを解決するためにギタっぷるは誕生しました。

# 使い方  
使い方はとても簡単です。まず、新規登録をします。そこからはギターを教えてもらうか、自分でサービスを考えて誰かにレッスンのをするかは、あなたの自由です。
